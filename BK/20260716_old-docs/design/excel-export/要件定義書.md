## 01_概要

| 項目 | 内容 |
| --- | --- |
| 背景・目的 | Excelで管理している社員の経歴書情報をWebアプリ化し、アプリ上で閲覧・編集ができるようにする。 |
| 対象ユーザー | 一般社員、管理職（部長など、部下の経歴書も編集できるユーザー）、システム管理者（マスタデータ・アカウントを管理するユーザー） |
| 課題 | 経歴書がExcelで個別管理されており、検索・横断的な把握（資格保有者は誰か、特定スキル保有者は誰か等）がしにくい |
| 解決後の姿 | Web上で経歴書を一元管理し、検索・横断表示ができる状態 |

## 02_機能要件

| 区分 | 機能 | 概要 |
| --- | --- | --- |
| 認証・権限 | ログイン | 社員ID＋パスワードでログインする |
| 認証・権限 | パスワード変更／リセット | メール送信によるリセットフロー |
| 認証・権限 | 権限体系 | 一般社員／管理職（部長など）／システム管理者の3階層 |
| プロフィール管理 | プロフィール（個人）閲覧・編集 | 本人が閲覧・編集する |
| プロフィール管理 | プロフィール一覧閲覧 | 全社員分を管理者・一般ユーザー共通で閲覧 |
| プロフィール管理 | プロフィール（メンバー）閲覧 | 他社員の経歴書閲覧のみ（編集不可） |
| プロフィール管理 | 保持項目 | 氏名、カナ、生年月日、性別、最寄り駅、経験年数、経歴概要、自己PR、最終学歴、保有スキル、保有資格、現場（プロジェクト）経歴 |
| 横断検索 | 資格保有者一覧 | 資格・氏名で検索 |
| 横断検索 | スキル保有者一覧 | 資格保有者一覧と同様の仕組みでスキル軸の横断検索を行う |
| マスタ管理 | アカウント管理 | 社員ID・パスワード・権限の登録編集削除 |
| マスタ管理 | スキルカテゴリ／スキル管理 | CRUD |
| マスタ管理 | 資格カテゴリ／資格管理 | CRUD |
| マスタ管理 | 駅・路線管理 | CRUD |
| マスタ管理 | 現場ポジション（役割）管理 | project_role（SE/PG/リーダー等の役割マスタ）のCRUD。プロジェクト自体のマスタ管理は対象外 |
| マスタ管理 | 部署管理 | 部署マスタ（Department）のCRUD、および各社員への部署割り当て |
| 入出力 | Excel取込 | 初期データ投入用の初回限定機能。継続的な一括更新機能としては持たない。ファイル形式はExcel（.xlsx） |
| 入出力 | 出力機能 | CSV/Excel出力機能は持たない |
| 非機能要件 | セキュリティ | パスワードはハッシュ化、権限に応じた画面・操作制御 |
| 非機能要件 | 画面デザイン | 見た目にこだわりたいという要望あり（優先度はフェーズ3） |

## 03_権限マトリクス

| 機能 | 一般 | 管理職 | システム管理者 |
| --- | --- | --- | --- |
| 自分の経歴書編集 | ○ | ○ | ○ |
| 他社員の経歴書閲覧 | ○ | ○ | ○ |
| 他社員の経歴書編集 | × | ○（自部署配下のみ） | ○（全社員） |
| 横断検索（資格保有者一覧・スキル保有者一覧等） | ○ | ○ | ○ |
| マスタ管理（スキル/資格/駅・路線/現場ポジション/部署） | × | × | ○ |
| アカウント管理 | × | × | ○ |

## 04_フェーズ分け

| フェーズ | 内容 |
| --- | --- |
| フェーズ1（MVP・ハッカソンのデモ中心） | ログイン、プロフィール一覧/詳細/編集（本人・自部署配下の編集含む）、部署マスタCRUD、スキル・資格マスタCRUD、資格保有者一覧・スキル保有者一覧 |
| フェーズ2 | Excel取込（初回投入限定機能として実装）、駅・路線／現場ポジションマスタ、パスワードリセット、アカウント管理、プロジェクト経歴の詳細管理 |
| フェーズ3 | 検索・絞り込み強化、ダッシュボード、画面デザインの作り込み |

## 05_決定事項

| 項目 | 決定内容 | ADR |
| --- | --- | --- |
| 管理職の編集可能範囲 | 自部署配下のみ（部署単位の判定、上長個人までは持たない） | 0002-manager-edit-scope-and-department.md |
| 社員IDの発番ルール | 単純連番（6桁ゼロ埋め、例:000001） | 0003-employee-id-format.md |
| 論理削除データの一覧表示方法 | 一覧はデフォルト非表示、フィルタ切替で表示可能にする | 0004-soft-delete-list-display.md |
| Excel取込の位置づけ | 初回投入限定（継続運用機能としては持たない）。形式はCSVではなくExcel（.xlsx） | 0005-csv-import-scope.md |
| 一覧のCSV/Excel出力機能 | 持たない（削除） | 0006-remove-export-feature.md |
