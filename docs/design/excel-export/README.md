## ファイル構成

- `要件定義書.xlsx` — 目次 + 01_概要 〜 06_未確定事項（7シート）
- `基本設計書.xlsx` — 目次 + 01_システム構成 〜 09_今後の課題（10シート）
- `詳細設計書.xlsx` — 目次 + 01_認証詳細 〜 10_未確定事項（11シート）。旧03〜06（画面詳細_ログイン／プロフィール編集／横断検索／マスタ管理共通）は項目列が共通のため`03_画面入力項目一覧`1シートに統合し、以降を04〜10に繰り上げ
- `DB設計書.xlsx` — 目次 + 01_共通カラム + 02_user 〜 19_project_skill_link + 20_ER図（21シート）。`docs/design/db-design.md`（Prisma向けに整理済みの物理名ベース定義）が元資料。20_ER図は箇条書きの文章から「起点テーブル／関係／終点テーブル／備考」の一覧表に組み直し済み
- `画面遷移図.xlsx` — 目次 + 01_画面遷移一覧（表） + 02_認証系 〜 04_マスタ管理系（遷移図3枚、画像埋め込み）。手作業で調整済みのため無変更
- `画面イメージ.xlsx` — 目次 + S-01〜S-20の20画面ぶんのワイヤーフレーム（Excelのセル罫線・塗りつぶしで表現したモックアップ）。手作業で調整済みのため無変更

各ブックの「目次」シートから各章シートへリンクで移動できる。表形式のシートは共通して以下を設定済み（画面遷移図・画面イメージの図版・モックアップシートを除く）。

- 見出し行の色付け・白文字・オートフィルタ・ウィンドウ枠固定
- 全セル薄グレーの罫線（表の境界を視認しやすくするため）
- データ行の1行おきの薄い網掛け（zebra striping、横読みしやすくするため）
- 日本語文書として読みやすいフォント（游ゴシック）に統一

画面一覧は`01_画面一覧.pdf`（22画面）を正とする（ADR `0007`、2026-07-08決定）。本ディレクトリのExcel・Markdownに含まれるS-01〜S-20の画面一覧は旧仕様（S-20 Excel取込は2026-07-06にADR `0005-csv-import-scope.md`の追記として追加されたが、ADR `0007`で画面としては廃止）。

## 更新方法

元のMarkdown（requirements.md / basic-design.md / db-design.md / detailed-design.md / screens.md）を更新した場合は、内容をこのExcelにも反映する（現時点では手作業での反映、または同等の生成スクリプトを再実行する）。詳細設計書の統合シート・DB設計書のER図表は`scripts`等に生成コードを残していないため、行を追加する場合は既存の書式（罫線・zebra・フォント）に合わせて手作業で追記する。
