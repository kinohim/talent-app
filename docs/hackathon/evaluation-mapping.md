# ハッカソン評価軸マッピング

## AI観点（メイン・50点、各10点）

| 評価項目 | 何を示せばよいか | 記録場所 |
|---|---|---|
| 設計工程 | 要件整理・仕様策定・アーキテクチャ/データモデル設計でのAI活用 | `docs/requirements/`, `docs/design/`, `docs/decisions/` |
| 実装工程 | コード生成・リファクタリング・デバッグでのAI活用 | `src/`, コミットログ, `docs/decisions/` |
| テスト工程 | テスト戦略・ケース設計・自動テスト生成 | `tests/`, `docs/decisions/` |
| インフラ | DB構築・デプロイ・監視などの運用基盤 | `infra/` |
| ハーネス | CLAUDE.md／サブエージェント／Skills／MCPなどの横断基盤 | `CLAUDE.md`, `docs/hackathon/agents-plan.md` |

## アイディア観点（補助・10点）
